🍽️ Rezept-Webseite
Hi, ich freue mich, dass Sie hier sind! 😊

Diese Webseite habe ich komplett mit JavaScript, HTML und CSS erstellt.
Ich habe das MVC-Architekturmuster verwendet und Axios genutzt, um Rezepte über eine API abzurufen. Außerdem kam Webpack zum Einsatz, um das Projekt zu strukturieren und zu bündeln.

Sie können Rezepte liken und später erneut verwenden – die gespeicherten Favoriten werden auf einem lokalen Server abgelegt.

Eine Übersicht über mögliche Suchbegriffe finden Sie hier:
👉 https://forkify-api.herokuapp.com/phrases.html

Ich hoffe, ich konnte damit ein kleines Beispiel meiner Fähigkeiten zeigen.
